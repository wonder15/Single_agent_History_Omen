# Single_agent_History_Omen
This script reproduces the full bet history of an agent [MUST PROVIDE MULTI-SIG ADDRESS] over a specified period of time (default 30 days).

INPUT: Agent multi-sig address, time config, API key for Omen subgraph.

OUTPUT: A table detailing bet history for a given agent of schema:
Date, Type (Buy/Sell), Bet (Pick| Yes/No), Result, Cost, Revenue, ROI, Trade ID.
